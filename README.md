## Selected Work

| Project | What it does | Stack |
|---|---|---|
| **[ML Doc Chatbot](https://github.com/L0adn9/ml-doc-transformer-chatbot)** | Answers ML tooling questions by sorting them into 15 topics. DistilBERT fine-tuned on 1,280 Q&A pairs — 97.3% val accuracy, 0.972 macro F1. Served as a Flask app. | `PyTorch` `Transformers` `Flask` |
| **[Tweet Sentiment Analysis](https://github.com/L0adn9/sentiment-analysis-twitter)** | Four approaches compared on 5,000 labelled tweets — TextBlob, VADER, TF-IDF + LogReg, LinearSVC — with an interactive Plotly dashboard. | `scikit-learn` `Plotly` `NLTK` |
| **[Traffic Sign Classifier](https://github.com/L0adn9/traffic-sign-classification)** | CNN for German traffic signs. Full pipeline: resize, normalise, augment, then accuracy/loss curves and a confusion matrix. | `TensorFlow` `Keras` `OpenCV` |
| **[Resume Parser](https://github.com/L0adn9/Resume_Parser_NLP)** | Pulls names, emails, skills and work history out of PDF/DOCX/TXT resumes into clean JSON. spaCy NER + regex + keyword matching. | `spaCy` `PDFMiner` `regex` |
| **[Movie Recommender](https://github.com/L0adn9/Movie-recommendation-system)** | Collaborative filtering built two ways — user-to-user and item-to-item cosine similarity, scored with RMSE. | `Pandas` `NumPy` `sklearn` |
